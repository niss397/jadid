Comment résoudre ?
Option 1 : Ouvrir dans Safari
1. Va dans l'app Fichiers sur ton iPhone
2. Long-press sur le fichier favoris-navigation-v9.html
3. Tape « Partager » → choisis Safari
4. Une fois ouvert dans Safari : « Partager » → « Sur l'écran d'accueil » pour l'avoir comme une app

Option 2 : Hébergement HTTPS gratuit (GitHub Pages)
Crée un repo GitHub public, active GitHub Pages, dépose le fichier, et ouvre l'URL https://... dans Safari.

01 Import depuis Apple Santé

Cet onglet importe ton historique cardio depuis l'app Santé de l'iPhone (alimentée par ton Apple Watch).
Note : Apple ne permet pas l'accès direct à HealthKit depuis une page web. Il faut donc faire un export manuel — c'est rapide.

📖 Comment exporter depuis Santé ? (1 min)
  1. Ouvre l'app Santé sur ton iPhone
  2. Tape ton avatar (en haut à droite)
  3. Tout en bas : « Exporter toutes les données santé »
  4. Confirme — l'iPhone prépare un fichier export.zip (peut prendre quelques minutes)
  5. Choisis « Enregistrer dans Fichiers » ou envoie-toi le fichier par mail
  6. Reviens ici, clique « Importer export.zip » ci-dessous

📋 Détails du contenu du backup
  • Favoris : tous tes lieux enregistrés (nom, adresse, GPS, catégorie, notes)
  • Catégories : tes catégories personnalisées et leur ordre
  • Promenades GPS : trajets avec points GPS, distance, durée, vitesse, altitude, couleur
  • Cardio : toutes les mesures importées (FC, HRV, SpO2, VO2 max, etc.)
  • Séances : workouts détectés (Apple Santé + .fit)
  • ECG : électrocardiogrammes Apple Watch
  • Paramètres : couleur navTrackColor, etc.

⚠ Le backup ne contient PAS les .zip Apple Santé ou .fit originaux — uniquement les données extraites.

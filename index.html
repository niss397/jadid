Comment résoudre ?
Option 1 : Ouvrir dans Safari
1. Va dans l'app Fichiers sur ton iPhone
2. Long-press sur le fichier favoris-navigation-v9.html
3. Tape « Partager » → choisis Safari
4. Une fois ouvert dans Safari : « Partager » → « Sur l'écran d'accueil » pour l'avoir comme une app

Option 2 : Hébergement HTTPS gratuit (GitHub Pages)
Crée un repo GitHub public, active GitHub Pages, dépose le fichier, et ouvre l'URL https://... dans Safari.
